Projects

Table Time Gaming

I built this project alone as part of the curriculum of DevMountain's web immersive development bootcamp. It is a board game forum and sharing site. Users can browse board games, comment to discuss the games, add a game to a favorites list, and suggest a game to be added. Each game page also includes a link to amazon with the search parameters for that game already entered. Admins can add new games to the list, remove games, and edit the game descriptions.

Social-Coders

I worked on this project in a group of four people as part of the curriculum of DevMountain's web immersive development bootcamp. My work on the project covered the full stack, but more especially on the back end. It is a social media platform for coders to connect, ask questions, and network with other like-minded individuals. Social Coders is a blend between facebook and stack overflow, with a reddit-esque layout and flow. This is an ongoing project but feel free to look around! If you want access to all the features, be sure to create an account and sign in.
